Here's your parametric OpenSCAD model — tweak any value and it regenerates:

// Spur gear · 24T · module 2

module gear(teeth=24, m=2, bore=8) {

pitch = teeth*m/2; outer = pitch + m;

linear_extrude(height=6)

difference(){ gear_shape(); circle(d=bore); }

}

gear(); // → 24T gear, ready to export
